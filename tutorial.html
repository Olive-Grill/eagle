Eaglercraft 1.8.8 Tutorial

Eaglercraft 1.8.8 is a copy of Minecraft 1.8.8 ported to HTML.

This tutorial will guide you on how to use Eaglercraft on eagle-plum.vercel.app.

Section 1: Singleplayer

Singleplayer mode works the same as Minecraft singleplayer. If you've never played Minecraft before, search for a "Minecraft 1.8" tutorial.

Section 2: Multiplayer

In Eaglercraft, you can play multiplayer locally or through an online server.

Local Play

To play locally, you must go into the singleplayer tab.

Setting Up a Local World

  • Create a world and press "Esc" to open the menu.
  • Click "Invite" and then "Continue" to adjust world settings.
  • Click "Start Shared World" and note the Relay URL and Join Code.

Joining a Local World

  • Go to "Multiplayer" from the home page.
  • Click "Network Settings" and set the correct Relay URL.
  • Click "Done," then "Cancel," then "Singleplayer."
  • Click "Join Shared World" and enter the code.

Online Servers

Background (Read Before Playing)

  • Servers use a username and password system.
  • Remember the username and password you use when registering.
  • Use a unique username if your first choice is taken.

Adding Servers

  • Go to "Multiplayer" and click "Add Server."
  • Enter a server address (must start with "wss://").
  • Examples:
    • ArchMC - wss://mc.arch.lol
    • Rice Network - wss://mc.ricenetwork.xyz
    • Vanilla MC - wss://vanillamc.me
    • Clever Teaching - wss://clever-teaching.com
    • Time Legacy - wss://mc.time-legacy.net
  • Press "Done," select the server, and click "Join Server."

Logging into Servers

  • Use /register [password] [password again] to register.
  • If your username is taken, change it in "Edit Profile."
  • On future logins, use /login [password] with the same username.

Section 3: Key Points

  • Worlds may be deleted if cookies are cleared.
  • Write down your username and password.
  • Use the same username/password combination for all servers.
